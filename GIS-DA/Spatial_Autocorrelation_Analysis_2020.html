Spatial Autocorrelation Analysis¶

Investigate spatial autocorrelation of higher education population is U.S. Counties following the material covered in class. Use the values for the number of persons over 25 in a county who have a graduate degree or higher. First, set any values that are not significantly different from 0 to zero (z - test using moe). Then -------->

  1. Determine weights (use Queen).
  2. Binary case: Number of persons over 25 with a graduate or professional degree exceeds the median county value.
    1. Join Counts
  3. Global Moran's I
  4. Hot and Cold spots (use 90% AND 95% confidence levels)
  5. Interpret the results.

Part 1: Read geographic data using Census FTP¶

This is an example of using the Census Bureau's FTP to extract county boundaries (spatial data) in contiguous U.S. (48 states). A parallel approach is to use the REST services. I have provided examples of both. If you are not familiar with Census Data and REST services, it may be easier to use the FTP site.

In addition, you will use the Census API to extract attribute data for analysis.

Reading Census geodata from the TigerWeb rest services involves examining the REST Map Services available, including the attributes and query tools. The URL below retrieves the GEOID, STATE, COUNTY, BASENAME, and polygon geometry of all states with a FIPS code < 57 in geojson format. The FIPS code for Alabama is 1 and for Wyoming is 56. This code may be used to eliminate terratories like Puerto Rico for the data set. Then Alaska (2) and Hawaii (15) from the dataset.

TIGER REST service https://tigerweb.geo.census.gov/arcgis/rest/services/TIGERweb/tigerWMS_ACS2019/MapServer/86/query?where=STATE%3C57&text=&objectIds=&time=&geometry=&geometryType=esriGeometryEnvelope&inSR=&spatialRel=esriSpatialRelIntersects&relationParam=&outFields=GEOID%2C+STATE%2CCOUNTY%2CBASENAME&returnGeometry=true&returnTrueCurves=false&maxAllowableOffset=&geometryPrecision=&outSR=&returnIdsOnly=false&returnCountOnly=false&orderByFields=&groupByFieldsForStatistics=&outStatistics=&returnZ=false&returnM=false&gdbVersion=&returnDistinctValues=false&resultOffset=&resultRecordCount=&queryByDistance=&returnExtentsOnly=false&datumTransformation=&parameterValues=&rangeValues=&f=geojson

Request and API key here

Use your key to get the following data for all counties: ACS 2018, 5 yr est.

Universe: Population 25 years and over in the United States (Educational Attainment)

  • B06009_006E Total Graduate or Professional Degree Est
  • B06009_006M Total Graduate or Professional Degree MOE

Example url = 'https://api.census.gov/data/2018/acs/acs5?get=B06009_006E,B06009_006M,NAME&for=county:*&key=YOURAPIKEY'

Before the tables are joined, create a new geoid field in the attribute table that combine the state and county fields.
